# 🚀 Wärmepumpen-Anmelde-Tool

Ein interaktiver Assistent für die Anmeldung von Wärmepumpen.

## Wie man es nutzt

1. Gehe zu [share.streamlit.io](https://share.streamlit.io)
2. Melde dich an
3. Gib dieses Repository ein
4. Klick Deploy!
